## 🌟 ¿Qué es Búsqueda por Profundidad - Tienda?

Es un sistema de gestión de inventario que implementa el algoritmo de **búsqueda por profundidad (DFS)** para explorar y encontrar productos en una estructura jerárquica de categorías. Ideal para tiendas con catálogos organizados por categorías y subcategorías.

### 💡 Características Principales

```
🔍 Búsqueda DFS            🏪 Gestión Completa       📊 Estructuras Jerárquicas
   • Exploración recursiva     • Agregar productos         • Categorías y subcategorías
   • Backtracking              • Eliminar productos        • Árbol de navegación
   • Traversal eficiente       • Actualizar stock          • Herencia de propiedades
   • Optimización O(V+E)       • Listar inventario         • Organización lógica

⚡ Algoritmos Puros        📈 Rendimiento            🎯 Precisión
   • Implementación DFS        • Búsqueda rápida           • Resultados exactos
   • Sin dependencias externas • Bajo consumo de memoria   • Sin falsos positivos
   • Código limpio y educativo • Escalable                 • Manejo de duplicados
```

## 🎮 Uso del Sistema

### Ejemplo Básico

```csharp
using Tienda;

// Crear instancia de la tienda
var tienda = new Tienda();

// Agregar productos por categoría
tienda.AgregarProducto("Electrónica", "Laptop Dell XPS");
tienda.AgregarProducto("Electrónica", "Mouse Logitech");
tienda.AgregarProducto("Ropa", "Camisa Polo");

}
```

## 🧠 Algoritmo DFS (Depth-First Search)

### Complejidades

| Métrica | Valor | Descripción |
|---------|-------|-------------|
| **Tiempo** | O(V + E) | V: vértices (categorías), E: aristas (relaciones) |
| **Espacio** | O(V) | Stack de recursión |
| **Óptimo** | No | Encuentra solución, no la mejor |
| **Completo** | Sí | Explora todo el árbol |

### Visualización del Algoritmo

```
          Tienda (Raíz)
         /      |      \
    Electrónica  Ropa   Hogar
       /    \      |       |
   Laptop  Mouse  Camisa  Lámpara

Orden de exploración DFS:
1. Tienda
2. Electrónica
3. Laptop ✓ (encuentra producto)
4. Mouse
5. Ropa
6. Camisa
7. Hogar
8. Lámpara
```

### Implementación del Algoritmo

```csharp
public List<Producto> BuscarPorProfundidad(string nombre)
{
    var resultados = new List<Producto>();
    
    void DFS(Categoria categoria)
    {
        // Buscar en productos de categoría actual
        foreach (var producto in categoria.Productos)
        {
            if (producto.Nombre.Contains(nombre))
                resultados.Add(producto);
        }
        
        // Explorar subcategorías recursivamente
        foreach (var subcategoria in categoria.Subcategorias)
        {
            DFS(subcategoria);
        }
    }
    
    // Iniciar desde la raíz
    DFS(categoriaRaiz);
    return resultados;
}
```

```csharp
// Prueba 1: Búsqueda exacta
var resultado = tienda.BuscarPorProfundidad("Laptop Dell XPS");
// Esperado: 1 resultado

var resultados = tienda.BuscarPorProfundidad("Laptop");
```
